LibraryApp

A simple iOS application that showcases a library of books with details such as title, author, number of pages, publish year, category, and summary. Users can view a list of books and select one to see more detailed information.

Features

List of books with basic information
Detail view for each book with additional information
Project Structure

Book.swift: Defines the Book struct containing book properties.
BookData.swift: Contains a sample list of books in the BookData struct.
ViewController.swift: Manages the list of books using UITableView.
BookDataVC.swift: Displays detailed information about the selected book.
