# 📚 LibraryApp

An iOS app for book lovers to explore a curated list of books. Dive into details like author, publication year, and summaries of popular classics!

## 🌟 Features

- **Book List View**: Browse a list of books with titles and authors displayed.
- **Detailed Book View**: Tap on any book to see detailed information, including a summary, category, and page count.

## 📂 Project Structure

- **`Book.swift`**: Model struct for book properties like title, author, and more.
- **`BookData.swift`**: Contains sample book data for easy testing.
- **`ViewController.swift`**: Displays the main book list using `UITableView`.
- **`BookDataVC.swift`**: Shows selected book details in a separate view.

## 🚀 Getting Started

### Prerequisites
- **iOS 13.0+**
- **Xcode 12.0+**

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/dgknctn/LibraryApp.git
   ```
2. Open the project in Xcode:
   ```bash
   cd LibraryApp
   open LibraryApp.xcodeproj
   ```
3. Run on an iOS Simulator or a connected device.
